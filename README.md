## Primeros comandos

```powershell
# Crear entorno
$ virtualenv .venv
# O
$ python -m venv .venv

# Activar entorno
$ .\.venv\Scripts\activate

# Instalar dependencias en el entorno
$ pip install -r .\requirements.txt
```

## Ejecutar aplicacion

```powershell
$ python .\application.py
```
